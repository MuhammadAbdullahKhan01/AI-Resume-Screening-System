# AI-Powered Resume Screening System

## Project Title
AI-Powered Resume Screening System

## Project Brief
The AI-Powered Resume Screening System is a web-based HR tool that automatically evaluates and ranks resumes according to job descriptions using Natural Language Processing (NLP). It analyzes candidate profiles, matches them with required skills and experience, and helps HR teams save time by identifying the most suitable applicants.

## Technologies
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **NLP Tools:** spaCy / HuggingFace Transformers
- **Database:** SQLite (for local testing)
- **Hosting:** GitHub / Render / PythonAnywhere
