# AI Resume Screening System

## Project Title
AI-Powered Resume Screening System

## Project Brief
An HR tool that automatically ranks resumes based on job description matching using Natural Language Processing (NLP). It helps recruiters identify top candidates efficiently and reduce manual screening time.

## Technologies
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **NLP Tools:** spaCy, / HuggingFace Transformers
- **Database:**  MySQL

## Type
AI + NLP + Web
